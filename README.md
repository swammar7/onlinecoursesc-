# Online Course Management System - C++

A C++ application demonstrating object-oriented programming concepts through an online course platform simulation.

## Features
- Four course types (Language, Software, Professional Development, Health & Fitness)
- Student enrollment with capacity limits
- Shopping cart with discount calculation
- Course rating system
- Interactive console menu

## Technologies
- C++ (Object-Oriented Programming)
- Inheritance & Polymorphism
- Operator Overloading
- Template Functions

## How to Run
```bash
g++ MainProgram01.cpp -o course_system
./course_system
